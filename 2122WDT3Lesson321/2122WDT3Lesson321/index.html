Rock Paper Scissor

This practical challenges students to use the logical && operator in conjunction with random numbers to create a Rock Paper Scissor game. In this game both the computer and the user when select between Rock Paper and Scissor. Below are the rules for winning.

  • Scissor beats Paper
  • Paper beats Rock
  • Rock beats Scissor
After each game, display each players choice as an image and also state who won.
